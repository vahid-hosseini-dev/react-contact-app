# Contact App

Simple React project built with Vite to manage contacts.

## Features
- Add, edit, and delete contacts
- Responsive design
- State management with React hooks

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev` to start the development server

## Technologies
- React
- Vite
- CSS
